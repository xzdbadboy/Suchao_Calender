# 苏超日历 2026 | Suchao Calendar 2026

2026年江苏省城市足球联赛（苏超）完整赛程日历，支持导入所有主流日历应用。

## 📅 快速开始

### 导入日历

直接使用 `SuCha2026Schedule.ics` 文件：

1. **苹果日历**：双击打开ICS文件或右键 → "打开方式" → 日历
2. **Google日历**：设置 → 导入与导出 → 选择ICS文件导入
3. **Outlook**：文件 → 打开 & 导出 → 导入
4. **其他应用**：支持标准ICS格式的日历应用都可用

### 功能特性

✅ **完整赛程** - 2026年全年78场常规赛（22轮）  
✅ **实时比分** - 已完成比赛标注最终得分  
✅ **球队队标** - 每场比赛显示对应球队官方队标  
✅ **城市文化** - 包含各队队徽的设计语言说明  
✅ **跨平台** - 兼容所有标准日历应用
